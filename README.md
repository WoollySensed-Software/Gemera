# Необходимое для установки
- **PySide6**
  ```cmd
  pip install PySide6==6.7.2
  ```
- **PySide6-Addons**
  ```cmd
  pip install PySide6-Addons==6.7.2
  ```
- **PySide6-Essentials**
  ```cmd
  pip install PySide6-Essentials==6.7.2
  ```
- **pyqtgraph**
  ```cmd
  pip install pyqtgraph==0.13.4
  ```
- **TOML**
  ```cmd
  pip install toml==0.10.2
  ```
- **WSS-ToolKit**
  ```cmd
  pip install WSS-ToolKit
  ```
